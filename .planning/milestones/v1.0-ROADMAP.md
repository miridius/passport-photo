# Roadmap: PassPhoto

## Overview

PassPhoto delivers a browser-based passport photo cropping tool in two phases. Phase 1 builds the complete cropping tool with digital export -- image loading, canvas with pan/zoom, guide overlay, spec panel, and downloadable cropped photos. Phase 2 adds print layout for home printing. After Phase 1, the tool is fully usable for its core purpose.

## Phase Details

### Phase 1: Cropping Tool + Digital Export
**Goal**: User can load a photo, position it within a guided crop frame, and download a correctly-sized cropped photo for passport submission
**Depends on**: Nothing (first phase)
**Requirements**: LOAD-01, CROP-01..07, CROP-09, UI-01..04, EXPD-01, EXPD-02, INFO-01, INFO-02, COMPAT-01, COMPAT-02
**Success Criteria** (what must be TRUE):
  1. User can select a photo from their device and see it displayed within a 35:45 aspect ratio crop frame with darkened overlay outside the frame
  2. User can pan (drag/keyboard) and zoom (scroll/pinch/keyboard) the photo to position it, with guide bands showing acceptable chin and crown zones
  3. A visual hint communicates that "crown" means top of skull, not top of hair
  4. Spec reference panel displays Australian passport photo requirements including child-specific notes
  5. User can download the cropped photo at native source resolution (1:1 crop, quality 1.0) and at digital submission specs (max 1200x1600 px, never upscale, quality 0.92, 70KB-3.5MB)
  6. Tool works on desktop Chrome/Firefox and Android Chrome without gesture conflicts or layout breakage
  7. Rulers and dimension annotations are visible outside the crop frame (not clipped by overflow:hidden)

### Phase 2: Print Layout
**Goal**: User can generate a print-ready 9x13cm sheet with 4 passport photos tiled in a 2x2 grid, cut guides delineating the cutting buffer zone, and preview modal -- all at native source resolution with DPI metadata
**Depends on**: Phase 1
**Requirements**: EXPP-01, EXPP-02, EXPP-03
**Success Criteria** (what must be TRUE):
  1. User can generate a print layout that tiles the cropped photo onto a 9x13cm sheet
  2. Print layout includes visible cut guides showing where to cut individual photos
  3. Print layout downloads as a JPEG with DPI metadata for accurate physical sizing
