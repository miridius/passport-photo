# Requirements: PassPhoto

**Defined:** 2026-03-07
**Core Value:** Correctly crop a photo so the face size and position fall within Australian passport photo spec ranges

## v1 Requirements

### Image Loading

- [x] **LOAD-01**: User can load a photo from their device via file picker

### Cropping & Interaction

- [x] **CROP-01**: Photo displays within a fixed 35:45 aspect ratio crop frame
- [x] **CROP-02**: User can pan the photo within the crop frame via mouse drag and touch drag
- [x] **CROP-03**: User can zoom the photo within the crop frame via scroll wheel and pinch zoom (3% per step for fine control)
- [x] **CROP-04**: Semi-transparent overlay darkens area outside the crop frame
- [x] **CROP-05**: Guide bands show acceptable chin zone as a shaded range (bottom of chin should fall within this band)
- [x] **CROP-06**: Guide bands show acceptable crown zone as a shaded range (top of head/skull should fall within this band)
- [x] **CROP-07**: Crown-vs-hair visual hint communicates that "crown" means top of skull, not top of hair
- [x] **CROP-09**: Keyboard controls for fine-grained pan/zoom (arrow keys: 2px normal, 10px with Shift; +/- keys: 3% normal, 10% with Shift)

### UI Layout

- [x] **UI-01**: mm ruler displayed outside (left of) the crop frame for visibility against any photo background
- [x] **UI-02**: 32mm/36mm dimension bracket annotations displayed outside (right of) the crop frame
- [x] **UI-03**: Guide bands (crown/chin) remain inside the crop frame as colored overlays
- [x] **UI-04**: mm ruler displayed below the crop frame (0-35mm horizontal scale), same style as vertical ruler, hidden on mobile (< 480px)

### Export — Digital Submission

- [x] **EXPD-01**: User can download cropped photo as JPEG at native source resolution (no resizing — just crop at 1:1 from source pixels, quality 1.0, preserve maximum available quality; 413x531 at 300 DPI is the minimum acceptable output)
- [x] **EXPD-02**: User can download a version scaled/compressed to meet Australian digital submission specs (JPEG, max 1200x1600 px — never upscale, quality 0.92 to stay within 70KB-3.5MB file size range)

### Export — Print Layout

- [x] **EXPP-01**: User can generate a print layout that tiles the cropped photo onto a 9x13cm sheet (standard DM Sofortfoto format)
- [x] **EXPP-02**: Print layout includes cut guides showing where to cut individual photos
- [x] **EXPP-03**: Print layout is downloadable as a JPEG image with DPI metadata for correct physical sizing

### Information

- [x] **INFO-01**: Spec reference panel shows Australian passport photo requirements (dimensions, face size range, position)
- [x] **INFO-02**: Child-specific notes visible (under-3: mouth may be open, some leniency on eyes)

### Compatibility

- [x] **COMPAT-01**: Tool works on desktop browsers (Chrome, Firefox)
- [x] **COMPAT-02**: Tool works on Android mobile browser (Chrome)

## v2 Requirements

- **LOAD-03**: Drag-and-drop photo loading
- **LOAD-04**: Camera capture directly from device

### Export

- **EXPD-03**: 600 DPI export option
- **EXPP-04**: A4 print layout option

### Interaction

- **CROP-08**: Undo/redo for pan/zoom adjustments

## Out of Scope

| Feature | Reason |
|---------|--------|
| Face detection / auto-positioning | Manual pan/zoom with guides is sufficient for this use case |
| iOS Safari / Safari support | Not needed — Chrome and Firefox only |
| Background removal / replacement | Out of scope — crop only |
| Photo editing (brightness, contrast, color) | Not a photo editor |
| Multiple country specs | Australia only |
| User accounts / cloud storage | Purely local tool |
| PDF export | JPEG print sheets work at DM; no need for PDF |
| Multi-photo session (batch processing) | Only 2 kids, one photo at a time is fine |
