Present the following verification checklist to the user:

**Desktop Chrome (COMPAT-01):**
1. Open http://localhost:5173 in Chrome
2. Click "Select a photo" and load a portrait photo
3. Verify: photo displays in 35:45 crop frame with darkened area outside
4. Verify: chin and crown guide bands are visible as colored regions
5. Verify: crown hint text is visible near the crown band
6. Drag to pan the photo — verify smooth movement
7. Scroll wheel to zoom — verify zoom works centered on cursor
8. Verify: spec panel shows Australian passport photo requirements
9. Verify: child-specific notes are visible (under-3 leniency)
10. Click "Download Print" — verify a PNG file downloads
11. Click "Download Digital" — verify a JPEG file downloads
12. Check downloaded print file: should be 413x531 pixels
13. Check downloaded digital file: should be 1200x1600 pixels, between 70KB and 3.5MB

**Desktop Firefox (COMPAT-01):**
14. Repeat steps 2-13 in Firefox
15. Pay special attention to export — Firefox may handle canvas.toBlob differently

**Android Chrome (COMPAT-02):**
16. Access dev server from Android device (use local network IP shown in terminal)
17. Tap "Select a photo" — verify file picker opens
18. After loading: touch-drag to pan — verify no page scroll/bounce conflict
19. Pinch to zoom — verify photo zooms, NOT the browser page
20. Verify guide bands and spec panel are visible (may need to scroll down)
21. Tap download buttons — verify files download to device
22. Verify layout is single-column and readable on mobile screen
