<feature>
  <name>Print Sheet Layout + Compositing</name>
  <files>src/lib/canvas/printSheet.ts, src/lib/canvas/printSheet.test.ts</files>
  <behavior>
    **calculateSheetLayout(cropWidthPx: number) -> SheetLayout:**
    - pxPerMm = cropWidthPx / 35 (photo width, not tile width)
    - sheetWidthPx = round(90 * pxPerMm), sheetHeightPx = round(130 * pxPerMm)
    - 4 tiles in 2x2 grid, centered on sheet with equal margins
    - marginX = (90 - 2*36 - 5) / 2 = 3.5mm, marginY = (130 - 2*46 - 5) / 2 = 16.5mm
    - Each tile: tileX/tileY at outer 36x46mm boundary, photoX/photoY at inner 35x45mm boundary (0.5mm inset)
    - Tile [0,0]: tileX = 3.5mm, tileY = 16.5mm, photoX = 4.0mm, photoY = 17.0mm
    - Tile [0,1]: tileX = 3.5 + 36 + 5 = 44.5mm
    - Tile [1,0]: tileY = 16.5 + 46 + 5 = 67.5mm
    - Tile [1,1]: tileX = 44.5mm, tileY = 67.5mm

    **drawCutMarks(ctx, tile, pxPerMm):**
    - 4 L-shaped marks per tile at photo corners (inner boundary)
    - Top-left: horizontal arm goes LEFT, vertical arm goes UP
    - Top-right: horizontal arm goes RIGHT, vertical arm goes UP
    - Bottom-left: horizontal arm goes LEFT, vertical arm goes DOWN
    - Bottom-right: horizontal arm goes RIGHT, vertical arm goes DOWN
    - Arm length = 2mm * pxPerMm
    - Line width = 0.6 * (pxPerMm / (300/25.4)) -- scales with resolution
    - Stroke color: #333333, lineCap: square

    **renderPrintSheet(img, crop, layout) -> Promise of Blob:**
    - Creates sheet canvas at layout dimensions, fills white
    - Renders crop to temp canvas once at photoWidthPx x photoHeightPx (native resolution)
    - Draws temp canvas 4x at tile photo positions
    - Draws cut marks for all 4 tiles
    - toBlob as JPEG quality 1.0
    - changeDpiBlob at 300 DPI
  </behavior>
  <implementation>
    1. Define SheetLayout and TilePosition interfaces
    2. Implement calculateSheetLayout as pure function (no DOM needed)
    3. Implement drawCutMarks using Canvas lineTo (helper drawLMark for each L)
    4. Implement renderPrintSheet: create sheet canvas, render crop to temp, tile 4x, add cut marks, toBlob + changeDpiBlob
    5. Import calculateSourceRect from export.ts for crop math (don't reimplement)
    6. Export all public functions and types

    This plan does NOT import from spec.ts -- it defines its own layout constants inline. This keeps it independent of Plan 01 and testable in isolation. Plan 03 will wire everything together.
  </implementation>
</feature>
